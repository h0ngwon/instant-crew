# Instant Crew

## 프로젝트 내용 
이용자가 관심사 카테고리(맛집, 문화예술 등)를 선택 하면 해당 카테고리 게시글들을 최신순으로 보여주고 등록할 수 있게 하여 이용자간의 상호작용을 목적으로 하는 어플리케이션<br/>

## 개발 일정
2023.12.26 - 2024.01.03<br/>
